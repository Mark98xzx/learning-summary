## 命名空间

在 `TS` 中，`export` 和 `import` 称为 外部模块，`TS` 中还支持一种内部模块 `namespace`，它的主要作用只是单纯的在文件内部（模块内容）隔离作用域

```typescript
    namespace k1 {
        let a = 10;
        export var obj = {
            a
        }
    }

    namespace k2 {
        let a = 20;
        console.log(k1.obj);
    }
```
