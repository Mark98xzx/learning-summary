- 近代形而上学（机械）唯物主义

- 物质的唯一特性是客观实在性，同旧唯物主义物质观划清了解下

> 客观实在性是**一切物质**的共性， 自然物质的共性

    - 运动 
    - 静止

### 第二节 物质世界的普遍联系和永恒发展

- 唯物辩证法

#### 2大特征

- 联系
    - 含义：事物之间以及事物内部诸要素之间相互影响、相互作用和相互制约
    - 客观性：事物本身固有，不以人的意志而转移
    - 普遍性：任何事物都不孤立地存在，都与其他事物处于相互联系之中，整个世界都是一个相互联系的整体
    - 多样性：多种多样的联系
- 发展（选择题或者简答题）
    - 含义：新事物代替旧事物的过程
    - 新事物：符合规律
    - 新事物必然战胜旧事物的原因
        1. **趋势**，新事物符合事物发展必然趋势（根本原因、标志）
        2. **优越**，新事物优越于旧事物
        3. **人民**，新事物符合广大人民群众的利益和要求，能得到广大人民群众的拥护和支持

    - 两种对立的发展观（选择题）
        - 联系 + 发展 => 辩证法   总特征
        - 孤立 + 静止 => 形而上学

#### 3大规律

- 对立统一规律（矛盾规律）（选择题或者简答题）
    - 含义：即矛盾规律，任何事物都包含着矛盾，矛盾双方既**统一**又**斗争**，由此**推动**事物运动变化发展。
    > 动力、源泉
    - 1 矛盾的基本属性：同一性和斗争性（选择题）
        - 统一 => 同一性 相互依存、转化
        - 斗争 => 斗争性 相互排斥
    - 2 内外因（选择题）
        - 内部矛盾 => 内因 => 根据
        - 外部矛盾 => 外因 => 条件
    - 内外因的辩证关系（简答题、论述题）
        1. 内因是事物发展变化的根据
        2. 外因是事物发展变化的条件
        3. 外因通过内因而引起作用
    - 对外开发 外因
    - 独立自主、自力更生 内因
    - 我们党制定独立自主、自力更生以及对外开放方针的理论基础是
        - 事物发展的内因和外因辩证关系的原理
    
    - 3 矛盾的普遍性和特殊性（选择题）
        - 任何事物 => 普遍性
        - 每个事物 => 特殊性 => 具体问题具体分析  eg：世界上没有包治百病的药方、一把钥匙开一把锁
    - 矛盾的普遍性和特殊性关系
        - 3.1 相互连结
            - 玉米、高粱、小麦 （特殊、个别、个性）
            - 以上都是 粮食 （普遍、一般、共性）
        - 3.2 相互区别
        - 3.3 相互转化

    - 方法论：马克思主义同中国实际相结合，建设中国特色社会主义
                普遍                       特殊性

    - 4 矛盾的不平衡性（选择题或者简答题）
        - 主要矛盾 （工作）
            - 人机关系  次要方面
            - 工作能力  主要方面
        - 次要矛盾 （生活）

        > 不同矛盾：**主要矛盾**和**次要矛盾**  **主矛和次矛**
        > 同一个矛盾：**主要方面**和**次要方面** **主方和次方** 

        - 方法论：两点论和重点论统一
            1. 两点论：二者不可偏颇，既要研究主要矛盾，又要研究次要矛盾，既要研究矛盾的主要方面，又要研究矛盾的次要方面
            2. 重点论：在研究复杂事物时，要着重地把握它的主要矛盾。
                - 在研究任何一种具体的矛盾时，要着重把握它的主要方面。

    ##### 对立统一规律（矛盾规律）地位及其原因（选择题或者简答题）
    
    - 地位： 对立统一规律是唯物辩证法的**实质与核心**，是**根本**规律。
    - 原因：
        1. 承不承认对立统一规律以及矛盾是事物发展的动力是唯物辩证法与形而上学斗争的**焦点**和**根本分歧**
        2. 唯物辩证法其他规律和范畴的**中心线索**
        3. 揭示普遍联系的根本内容和事物发展变化的**内在动力**
        4. **矛盾分析法**是最根本的认识方法

- 质量互变规律
    - 量变、质变和度（选择题）
        - 量变：事物数量（发展的规模/程度/速度等）的增减，没有发生根本性变化
        - 质变：事物发生根本的变化，事物又一种质态向另外一种质态的飞跃
        - 度：界限
    1. 区分量变和质变的根本区别是事物的变化是否突破（度）
    2. 揭示事物发展状态和形式的规律

    - 量变和质变的辩证关系（选择题或者简答题）
        - 量变是质变的必要准备
        - 质变是量变给必然结果

- 否定之否定规律

    - 三大规律之否定之否定规律（辩证的否定观）
        - 批判 + 继承
    - 形式：螺旋式上升和波浪式前进，揭示了事物发展是**前进性**和**曲折性**的统一，揭示了事物发展的**趋势**和**道路**

    - 对**传统文化**的批判继承态度依据的是**辩证的否定**的原理

#### 5对范畴

- 原因VS结果 （选择题）
    - 含义： 
        - 原因：引起某种现象的现象
        - 结果：被某种现象所引起的现象
    - 关系
        - 对立统一
            1. 原因和结果的对立的
            2. 相互依存和在一定情况下相互转化
            3. 原因和结果是相互作用的
    
- 必然性VS偶然性
    - 含义：
        1. 必然性：事物联系和发展过程中**确定不移的、不可避免**的趋势
        2. 偶然性：**不确定**的趋势
    - 关系： **对立统一**，相互作用

    > 只看到必然，否定偶然存在 -- 机械决定论

- 可能性VS现实性
    - 含义：
        1. 现实性：客观事物和现象的种种联系（已发生）
        2. 可能性：包含在现实事物之中、预示着事物发展前途的种种趋势（未发生）
    - 关系
        - 对立统一

- 内容VS形式
    - 含义：
        1. 内容：构成事物一切**要素**的总和     内在
        2. 形式：把内容各要素统一在一起的**结构**或表现形式
    - 关系：
        1. 对立：内容和形式含义不同
        2. 统一：相互作用

- 本质VS现象
    - 含义：
        1. 本质：事物的**根本性质**以及组成事物基本要素的内在联系
        2. 现象：事物的**表面特征**以及这些特征的外部联系
    - 关系：
        1. 对立：现象是外部，本质的内部；现象是多变，本质稳定
        2. 统一：本质都有通过一定的现象表现出来
